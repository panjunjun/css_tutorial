MLR的模式完全行得通：作者免费投稿，审稿人义务审稿，读者免费下载，数据库由学生volunteer维护。这是一个良性循环，减少学者科研经费的负担，减少学校购买期刊的开支，让大众可以免费获取科研论文。我投过很多篇，没有花钱；审过很多篇JMLR，也没收钱；我下载期刊论文也没花钱。我没看出这种模式有任何坏处，想不出想不出任何理由让出版商从学术期刊中牟取利益，也不赞同学术期刊只对购买数据库的机构开放。JMLR写得很清楚，版权属于作者自己；而大多数的期刊版权属于出版商。人类智慧的结晶，凭什么被出版商据为己有？我白天在学校看IEEE各种期刊会议论文很容易，晚上在家全都点不开，想看得花几十美元。这种感觉无异于在中国上Google。Machine Learning已经有JMLR、MLJ、PAMI等很多期刊。Nature搞Machine Intelligence期刊还不是看到有利可图？人家可不是搞慈善的！当年那个不开放的顶级期刊MLJ已经没落了，没道理让这个开倒车的期刊活下去。只要一流的学者不给不审稿，这个期刊注定成为二流（也不见得是坏事，以后大家想发nature子刊倒是容易多了）。所以我也签名了，坚决不投稿、不给这个期刊审稿。
